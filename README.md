# JSON Assignment

## 📌 Project Overview
This project demonstrates how to work with JSON in Python using the built-in `json` module.

It includes:
- Creating JSON structures
- Converting Python dictionaries to JSON strings
- Parsing API responses
- Handling nested JSON data

---

## 🛠 Technologies Used
- Python 3.x
- Built-in `json` module

---

## 📂 Tasks Completed

### ✅ Task 1 — Build a JSON Structure
- Created a user profile dictionary
- Converted it to formatted JSON using `json.dumps()`

### ✅ Task 2 — Parse an API Response
- Parsed a JSON string using `json.loads()`
- Extracted username and score
- Printed formatted message:

User alex99 scored 87.5 points


### ✅ Task 3 — Handle Nested JSON
- Extracted city and zip code
- Added `"country": "India"` inside address
- Printed updated JSON
